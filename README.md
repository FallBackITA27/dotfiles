# dotfiles

## nvim
Prerequisites:

1. Install Rust Analyzer
```bash
cargo component add rust-analyzer
```
2. Within nvim, with :Mason, install the following list:
    1. `clangd`
    2. `css-lsp`
    3. `emmet-language-server`
    4. `html-lsp`
    5. `stylua`
    6. `typescript-language-server`

## sway
For screenshots to work, install `flameshot`
```bash
sudo pacman -S flameshot
```
